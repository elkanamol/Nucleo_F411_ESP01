# Nucleo_F411_ESP01

This App demonstrate Nucleo_F411RE  connecting to ESP01  (ESP8266) through UART and get enviroment data by BME280 over I2C bus. 
the unit send the data to ThingSpeak cloud and persent it by three chatrs.
